# StarShip — Собери витамины!

Мобильная веб-игра: космический корабль собирает витамины Супрадина, избегая мусора.

## Запуск

**Не открывайте `index.html` напрямую** — браузер блокирует ES‑модули по `file://`. Нужен локальный сервер:

```bash
npm run serve
# или
npx serve . -l 3000
# или
python3 -m http.server 3000
```

Откройте **http://localhost:3000**

## Структура

- `index.html` — разметка
- `css/` — стили
- `js/` — модули игры
- `docs/PLAN.md` — план развития
